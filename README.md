# WebSecScanner

Web uygulamaları için güvenlik tarama aracı. SQL Injection, XSS, Command Injection gibi yaygın zafiyetleri tespit eder.

## Özellikler

- SQL Injection testi
- XSS testi
- Command Injection testi
- Çoklu hedef tarama
- Form ve URL parametrelerini otomatik tespit
- JSON, CSV, HTML, TXT rapor çıktısı
- Proxy desteği
- Rate limiting / delay
- Renkli terminal çıktısı
- Detaylı logging

## Kurulum

```bash
# Repoyu klonla
git clone https://github.com/yourusername/websecscanner.git
cd websecscanner

# Gereksinimleri yükle
pip install -r requirements.txt

# veya direkt kurulum
python setup.py install
